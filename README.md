# JoshuaHouseDev

My portfolio website built with React, Vite, Bootstrap, and WebGL shaders.

## Development Setup

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── assets/         # Assets and images
├── components/     # React components
├── shaders/        # WebGL shaders (GLSL)
├── main.jsx        # App entry point
├── index.css       # Global styles
```
