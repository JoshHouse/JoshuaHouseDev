# JoshuaHouseDev

My portfolio website built with React, Vite, Bootstrap, and WebGL shaders.

## Development Setup

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── assets/         # Assets and Images
├── components/
    ├── CSS/        # React Component Styling
    ├── JSX/        # React Component JSX Files
├── shaders/        # WebGL Shaders (GLSL)
├── main.jsx        # App Entry Point
├── main.css        # Global Styles
```
