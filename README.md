# JoshuaHouseDev

My portfolio website built with React, Vite, Bootstrap, and jQuery.
Domain name obtained from GoDaddy.com
Hosted through GitHub Pages

## Development Setup

Install Requirements:
```bash
npm install
```

Run Local Development Enviornment
```bash
npm run dev
```

## Project Structure

```
src/                         # Contains Main.jsx and Main.css
├── assets/                  # Assets and Images
    ├── images/              # Images Used Throughout the Pages
    ├── videos/              # .mp4 Videos (Not Used and Replaced with YouTube Embeds)
├── components/
    ├── CSS/                 # React Component Styling (For Styles that Couldn't be Done with Bootstrap)
    ├── JSX/                 # React Main Pages Component JSX Files
        ├── ProjectPages/    # React Project Pages Component JSX Files
├── shaders/                 # WebGL Shaders (GLSL) Setup (Didn't End Up Being Used)
```
